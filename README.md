# Solana-course
# Proyecto Solana - Contador

Este proyecto es un ejemplo simple en la blockchain de Solana usando Anchor.

## Función
El programa crea un contador y permite incrementarlo.

## Tecnologías
- Solana
- Rust
- Anchor Framework
- TypeScript
- Solana Playground

## Pasos para ejecutarlo

1. Abrir https://beta.solpg.io
2. Crear proyecto Anchor
3. Pegar el código en src/lib.rs
4. Build
5. Deploy
6. Ejecutar client.ts
